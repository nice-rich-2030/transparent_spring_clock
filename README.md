# Transparent Spring Clock

透明なスプリングドライブ式時計の3Dビューアです。three.jsを使用してリアルタイムに描画される時計のムーブメントを、あらゆる角度から観察できます。

## 特徴

- **リアルタイムJST表示** — 日本標準時をスイープ運針で表示
- **透明ムーブメント** — 全パーツが半透明で内部構造が見える
- **インタラクティブ操作** — マウスドラッグで自由に回転・ズーム
- **パーツ情報表示** — ホバーで各部品の名称と説明をツールチップ表示

## 時計の構造

香箱車 → 2番車 → 3番車 → 4番車 → ガンギ車 → アンクル → テンプの輪列で動力を伝達。各車には外側の歯車とカナ（ピニオン）があり、前段の外歯が次段のカナと噛み合って回転を伝えます。

## 技術スタック

- [three.js](https://threejs.org/) (r160) — 3Dレンダリング
- [TypeScript](https://www.typescriptlang.org/) — 型安全な開発
- [Vite](https://vitejs.dev/) — 高速ビルドツール

## 開発

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

## 操作方法

| 操作 | 説明 |
|------|------|
| マウスドラッグ | 視点の回転 |
| スクロール | ズームイン/アウト |
| 右ドラッグ | パン移動 |
| ヘッダーメニュー | カメラプリセット切替 |
| フッターメニュー | 自動回転・ヘルプ |
